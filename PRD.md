
# 2026 공과대학 개강총회: 공대총회식 티켓팅 사이트 설계 명세

## 1. 프로젝트 개요

### 1-1. 서비스명

공과대학 개강총회: 공대총회식 티켓팅 사이트

### 1-2. 목적

공과대학 재학생을 대상으로 행사 참가자를 **선착순 모집**하고,
참가자에게 **좌석 선택 경험**을 제공하며,
운영진이 신청 현황을 쉽게 관리할 수 있도록 하는 웹 기반 신청 시스템 구축.

### 1-3. 핵심 목표

* 정해진 시각에 맞춰 선착순 신청 진행
* 좌석 번호 기반 신청
* 중복 신청 방지
* 신청자 정보 수집
* 운영진용 참가자 관리 기능 제공

# 5. 좌석 시스템 설계

## 5-1. 좌석 구조

* 총 좌석 수: 약 100석
* 좌석 번호 예시:

  * A1~A20
  * B1~B20
  * C1~C20
  * D1~D20
  * E1~E20

또는 실제 술집 구조를 반영하여

* 1번 테이블 1석
* 1번 테이블 2석
* 2번 테이블 1석
  형식으로 세분화 가능

## 5-2. 좌석 선택 정책

* 사용자는 한 번에 한 좌석만 선택 가능
* 좌석 선택 시 3~5분 임시 홀드
* 결제 기능이 없다면 3분 홀드 권장
* 신청 제출 완료 시 좌석 확정

## 5-3. 실시간 동기화

* 다른 사용자가 이미 선택한 좌석은 즉시 비활성화
* 좌석 충돌 시 먼저 제출 완료한 사용자 우선
